Healthy food inquiry

Description
This inquiry is designed to be used to make virtual experiments online, based on data collected from thermal swimming pools in a real setting. For example, users of this inquiry could observe buoyancy in serveral thermal bath pools of different temperature with different minerals. Thus, experiment designers think out different means and methods with limited existing resources arround the swimming pools.
